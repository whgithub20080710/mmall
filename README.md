## 电商项目，使用tomcat集群和redis分布式。

## 相关内容：

- Lombok使用
- Maven多环境隔离
- Tomcat集群部署
- 使用Redis实现单点登录功能
- Redis连接池封装(单Redis和Redis分片)
- JsonUtil封装(基于Jackson)
- SpringSession集成
- SpringMVC全局异常处理(HandlerExceptionResolver)
- SpringMVC 拦截器实现权限统一校验
- Spring Schedule实现定时关单
- Spring Schedule+Redis分布式锁构建分布式任务调度
- Redisson框架集成
- logback热加载
